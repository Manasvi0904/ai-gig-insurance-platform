
# AI-Powered Parametric Insurance for Gig Workers

📌 Problem Statement
Gig workers (Zomato, Swiggy, Zepto, etc.) lose 20–30% income due to external disruptions like heavy rain, heatwaves, pollution, or curfews.

Currently, there is NO protection for income loss.

Our solution provides AI-powered parametric insurance with automatic payouts based on real-world triggers.

📌 Target Persona
We focus on Food Delivery Partners (Swiggy/Zomato riders).

Pain Points:
- Income depends on daily working hours
- Weather disruptions reduce delivery capacity
- No financial safety net

📌 Solution Overview
We propose a parametric insurance platform that:

- Tracks real-time environmental conditions
- Detects disruptions automatically
- Calculates income loss
- Instantly pays compensation

📌 Weekly Premium Model
Premium is calculated weekly based on:

- Avg weekly income
- Risk level (weather, location)
- Working hours

Example:
If weekly income = ₹4000
Risk factor = Medium

Premium = ₹50–₹80/week

Higher risk zones → higher premium

📌 Parametric Triggers
Payout is triggered automatically when:

- Rainfall > threshold (e.g., 50mm/day)
- Temperature > 42°C
- AQI > 300
- Govt curfew/zone shutdown

No manual claim required

📌 AI/ML Integration 🤖
1. Risk Prediction Model
   - Predicts probability of disruption
   - Adjusts weekly premium dynamically

2. Fraud Detection
   - Detects fake claims using:
     - GPS mismatch
     - Activity logs
     - Duplicate claims

3. Income Loss Estimation
   - Predicts expected vs actual earnings

📌 System Workflow
1. User signs up
2. AI calculates risk profile
3. Weekly premium is assigned
4. System monitors real-time data (weather, etc.)
5. If disruption detected → payout triggered
6. Money transferred instantly
